커맨드 팔레트에서 (커맨드 + shift + P)를 누르고 shell command 라고 입력한 다음 위의 Install 'code' command in PATH에 입력하면 자동으로 설치가 된다.

터미널에서

```shell
code
```

라고 입력하면 VS Code의 기본 창이 뜨게 되고

```shell
code my-project
```

라고 입력하면 프로젝트가 열리게 된다.


혹은

```shell
cd my-project 
code .
```

도 된다.
